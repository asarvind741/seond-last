1) ssh username@SERVER_IP_ADDRESS
2) if you want to connect to specific port then use ssh username@SERVER_IP_ADDRESS -p <port>
3) if you are trying to connect to server using a private key then run below command-
   ssh <your-user-name>@<server-public-IP-address> -p <port-number> -i <your-local-ssh-private-key-name>
4) if are connecting first time it will give a warning.
5) You can just time "YES" and continue.
6) It will ask you to enter password. Please do that.
7) run the command "ls" and see list of directory.
